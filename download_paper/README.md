# download_paper

下载论文的python脚本，打算按关键字寻找某个会议的论文，然后批量下载。
目前可以通过`https://dblp.org`网站，按关键字批量下载VLDB和SIGMOD的论文。

## 思路

- 通过`https://dblp.org`网站，找到特定年份特定会议的论文下载地址；
- 再从对应的下载地址中找到论文名字和pdf的下载链接下载相应论文。
